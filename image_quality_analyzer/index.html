影像分析指標說明

PSNR (峰值訊噪比)

利用對數分貝 (dB) 表示兩圖像素差異。數值越高代表失真越少。大於 40 dB 在人眼視覺上即為無損等級。

SSIM (結構相似性)

高度符合人類視覺特性的感知指標，範圍為 0 ~ 1。大於 0.98 代表影像結構、亮度和對比度幾乎完全相同。

MSE (均方誤差)

計算兩圖對應像素通道數值差異的均方值。數值越低越接近原圖（若 MSE 為 0 代表兩圖完全相同）。
